ANIMATION

 Animation - To animate CSS elements

 Syntax:  @keyframes ---animation_name--- {
               from{
                    ---elments in orginal form---
                }
                to{
                    ---elments in final form after animation---
                }
           }
Animation Properties
Sl No. Properties Usage
1 "animation-name" It is used to set the name of the animation frame
2 "animation-duration" It is used to set the amount of time the animation will occur
3 "animation-timing-function" It is used to set the way the animation will happen
4 "animation-delay" It is used to set the amount of delay in animation
5 "animation-iteration-count" It is used to set the no.of times the animation will occur(1 to infinity)
6 "animation-direction" It is used to set whether an animation should play forward, backward, or alternate back and forth between playing the sequence forward and backward.
